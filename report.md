# Student Task Manager - Project Report

## 1. Introduction
In today’s fast-paced academic environment, students often struggle to manage their daily tasks effectively. This can lead to missed deadlines, poor time management, and reduced productivity. 

The Student Task Manager is a simple command-line application developed using Java to help users organize and manage their tasks efficiently. The system provides an easy-to-use interface for adding, viewing, and deleting tasks.

---

## 2. Problem Statement
Students frequently face difficulty in tracking their academic and personal tasks. Existing tools are often too complex or distracting, making them less suitable for quick and efficient task management.

This project aims to provide a lightweight and user-friendly solution that allows users to manage tasks effectively using basic programming concepts.

---

## 3. Objective
- To develop a simple task management system using Java  
- To apply Object-Oriented Programming (OOP) concepts  
- To implement file handling for data storage  
- To create an interactive command-line interface  

---

## 4. Methodology
The project is developed using the Java programming language. It follows Object-Oriented Programming principles by using classes and objects to structure the application.

An ArrayList is used to store tasks during runtime. File handling is implemented to save tasks into a text file (`tasks.txt`) so that the data persists even after the program is closed.

---

## 5. Implementation
The project consists of the following components:

- **Task Class**: Stores task ID and description  
- **Main Class**: Handles user interaction and program logic  
- **ArrayList**: Used to manage the list of tasks  
- **File Handling**: Used to save and load tasks from a file  

### Functionalities:
- Add new tasks  
- View all tasks  
- Delete tasks by ID  
- Save tasks automatically  
- Load tasks when the program starts  

---

## 6. Challenges Faced
- Handling invalid user input (non-numeric values)  
- Managing file reading and writing operations  
- Ensuring task IDs remain consistent after deletion  

---

## 7. Results
The application successfully allows users to manage their tasks efficiently. Tasks are stored permanently using file handling, ensuring data is not lost between sessions.

The system is simple, functional, and demonstrates key programming concepts effectively.

---

## 8. Conclusion
The Student Task Manager provides a practical solution to the problem of task management. It demonstrates the use of core Java concepts such as OOP, data structures, and file handling in a real-world application.

---
