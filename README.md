# Student Task Manager

## 📌 Description
A simple command-line based task manager built using Java. It helps users manage daily tasks efficiently.

## 🚀 Features
- Add tasks
- View tasks
- Delete tasks
- Save tasks using file handling

## 🛠️ Technologies Used
- Java
- OOP Concepts
- File Handling

## ▶️ How to Run
1. Open the project in any Java IDE (IntelliJ / Eclipse)
2. Run Main.java
3. Follow the menu instructions

## 📂 Project Structure
- Main.java → Main program file
- tasks.txt → Stores saved tasks
